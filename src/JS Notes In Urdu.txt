Variables wo dabba hota hai jisme hum apni values (jaise numbers, words) rakhte hain.

Example:
let name = "Ali";  // name variable mein "Ali" rakha
let age = 10;      // age variable mein 10 rakha
Yahan let se variable banate hain, aur usmein koi value rakhte hain.

Interview Question on Variables:
Q: JavaScript mein variable kya hota hai?
A: Variable ek container hota hai jisme hum data ko temporarily store karte hain.

Q: JavaScript mein variables declare karne ke kitne tareeqe hain?
A: Teen tareeqe hain: var, let, aur const.

Q: let aur const mein kya farq hai?
A:

let se banaya gaya variable ki value badli ja sakti hai.

const se banaya gaya variable ki value nahi badli ja sakti (constant rehta hai).

JavaScript mein mukhtalif qisam ke data hotay hain. Kuch important data types yeh hain:

String: Alphabets ya words. Jaise: "Hello", "Ali"

Number: Numbers jaise 10, 3.14

Boolean: Sirf do values le sakta hai: true (sahi) ya false (galat)

Undefined: Jab variable ko koi value nahi di gai

Null: Khas tor par "khaali" value dikhane ke liye
